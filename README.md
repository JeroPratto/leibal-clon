# Leibal

## How to install and Run

```bash
# Clone this project
git clone https://github.com/JeroPratto/leibal-clon.git
# Install
npm install
# Run
npm run dev
```

## Technologies

- Vue.js
- Vite
- Pinia
- Typescript
- Jest
- Vitest

## This is the final result.

Open [http://localhost:5173/](http://localhost:5173/) with your browser to see the result.
